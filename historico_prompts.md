# Histórico de Prompts do Projeto "Organizador Inteligente"

Este arquivo documenta as instruções e solicitações feitas para a construção e configuração deste projeto.

---

## Prompt 1: Criação do Programa Principal

> Você é um desenvolvedor Python especializado em automação de arquivos e criação de interfaces simples para desktop.
> 
> Quero criar um programa chamado “Organizador inteligente”, cujo objetivo é organizar automaticamente arquivos de uma pasta escolhida pelo usuário.
> 
> O programa deve ser desenvolvido em Python e possuir uma interface gráfica simples utilizando Tkinter ou CustomTkinter.
> 
> Funcionalidades obrigatórias:
> 
> * O usuário deve selecionar uma pasta do computador.
> * O programa deve identificar automaticamente os tipos de arquivos presentes na pasta.
> * Os arquivos devem ser movidos para subpastas organizadas por categoria, como:
> 
>   * Imagens
>   * PDFs
>   * Vídeos
>   * Documentos
>   * Planilhas
>   * Compactados
>   * Músicas
>   * Outros
> * Caso a subpasta não exista, o programa deve criá-la automaticamente.
> * O sistema deve evitar erros caso existam arquivos com nomes repetidos.
> * O programa deve mostrar uma mensagem final informando:
> 
>   * Quantos arquivos foram organizados
>   * Quantas categorias foram criadas
>   * Se ocorreu algum erro durante o processo
> 
> Requisitos técnicos:
> 
> * Utilize boas práticas de programação.
> * Separe o código em funções bem organizadas.
> * Comente as partes principais do código para facilitar entendimento.
> * Utilize as bibliotecas:
> 
>   * os
>   * shutil
>   * tkinter ou customtkinter
> * O código deve ser simples o suficiente para estudantes iniciantes entenderem.
> * Explique rapidamente a lógica de funcionamento antes de mostrar o código.
> 
> Também quero sugestões de:
> 
> * melhorias futuras,
> * novas funcionalidades,
> * possíveis otimizações para o sistema.
> 
> Por fim, gere:
> 
> 1. O código completo
> 2. Explicação resumida do funcionamento
> 3. Estrutura das pastas criadas
> 4. Sugestões de interface visual
> 5. Possíveis problemas e soluções

## Prompt 2: Configuração de Ambiente e Versionamento

> Crie um ambiente virtual (venv) para o projeto e instale automaticamente todas as bibliotecas necessárias dentro dele. Gere também um arquivo requirements.txt contendo todas as dependências utilizadas no projeto e configure um arquivo .gitignore apropriado para projetos Python, ignorando arquivos temporários, cache, ambiente virtual (venv), configurações locais e demais arquivos desnecessários para versionamento no Git.
